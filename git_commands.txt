classGit is a version control system used for tracking changes in source code. Here are some essential Git commands you should focus on:Basic Git Commands1. git init – Initialize a new Git repository.2. git clone <repository_url> – Clone an existing repository.3. git status – Check the status of your working directory.4. git add <file> – Stage a file for commit.5. git commit -m "message" – Commit changes with a message.6. git push origin <branch> – Push commits to a remote repository.7. git pull origin <branch> – Fetch and merge changes from the remote repository.8. git log – View commit history.Branching & Merging9. git branch – List all branches.10. git branch <branch-name> – Create a new branch.11. git checkout <branch> – Switch to a branch.12. git checkout -b <branch> – Create and switch to a new branch.13. git merge <branch> – Merge a branch into the current branch.14. git rebase <branch> – Reapply commits on top of another base branch.Undoing Changes15. git reset --hard <commit> – Reset the working directory to a specific commit.16. git reset --soft <commit> – Undo commits but keep changes staged.17. git revert <commit> – Create a new commit that undoes a previous commit.18. git stash – Save changes temporarily.19. git stash pop – Restore stashed changes.Remote Repositories20. git remote -v – View remote repositories.21. git remote add <name> <url> – Add a remote repository.22. git fetch – Fetch changes from a remote repository.23. git pull --rebase – Pull with rebase instead of merge.Advanced Commands24. git cherry-pick <commit> – Apply a specific commit from another branch.25. git tag <tag-name> – Create a new tag.26. git bisect – Find the commit that introduced a bug.
